## Unit Breakdown

## Getting Started

1. Clone or download the repository
2. Navigate to the unit you want to work on
3. Open the exercise description file (e.g., `exercises_1.txt`)
4. Create a new `.cpp` file for your solution
5. Implement the exercise according to the specifications
6. Compile and test your code

## Progression Path

These exercises are designed to be completed in order:
- **Beginner:** Units 1-2 (basic syntax and logic)
- **Intermediate:** Units 3-4 (functions and data structures)
- **Advanced:** Units 5-6 (complex data structures and file handling)

## Requirements

- C++ compiler (g++, clang, MSVC, etc.)
- Text editor or IDE (VS Code, Visual Studio, Code::Blocks, etc.)
- Basic command line familiarity

## Tips for Success

- Read exercise descriptions carefully before coding
- Start with simpler exercises and progress at your own pace
- Test your code with multiple input cases
- Review your code for clarity and efficiency
- Experiment with different approaches to problems

### Unit 1: Basic I/O and Simple Programs
Introduction to C++ with fundamental concepts like variables, input/output, and basic calculations.

**Topics:**
- Program structure and compilation
- Console input/output
- Variable types and declarations
- Basic arithmetic and string operations

**Key Exercises:**
1. Mailing Address - Print formatted output
2. Hello - User input and string handling
3. Area of a Room - Floating point calculations
4. Tax and Tip - Multi-step calculations with formatting
5. Odd or Even - Modulo operator
6. Sum of First n Positive Integers - Number series
7. Fahrenheit to Celsius - Temperature conversion
8. Bonus Exercise - Extended challenge

### Unit 2: Conditionals and Decision Making
Master if/else statements and conditional logic for program flow control.

**Topics:**
- if/else statements
- Nested conditionals
- Pattern matching with switch statements
- Multiple conditions with logical operators

**Key Exercises:**
1. Vowel or Consonant - Character classification
2. Faces on Money - Lookup tables with conditionals
3. Richter Scale - Range-based classification

### Unit 3: Functions and Code Organization
Learn to write reusable code through functions, with focus on parameters and return values.

**Topics:**
- Function definition and declaration
- Parameters and return types
- Function scope and visibility
- Code modularity and reusability

**Key Exercises:**
1. Random Password Generator - Random character generation
2. Password Validator - String analysis and validation
3. Random Good Password - Combining multiple functions
4. Caesar Cipher - Text transformation and encryption
5. Text Search - String matching and searching

### Unit 4: Data Structures and Collections
Work with arrays and lists for handling multiple data items.

**Topics:**
- Arrays and lists
- Iteration and loops
- Aggregation (sum, min, max, average)
- List manipulation

**Key Exercises:**
1. Sum Items in List - Calculating totals
2. Get Largest/Smallest Numbers - Finding extremes
3. String Length Calculations - Average values
4. Random List Generation - Creating datasets
5. String Pattern Matching - Counting elements

## Repository

### Download as ZIP

- On GitHub, open the repository page and click the green **Code** button, then select **Download ZIP**. Extract the ZIP to a folder on your machine.
- Alternatively, use `curl` or `wget` with the repository's `archive/refs/heads/main.zip` URL (replace `main` if the default branch differs).

### Git clone

- To clone the repository locally, run:

```
git clone https://github.com/<owner>/<repo>.git
```

- Replace `<owner>` and `<repo>` with the repository owner and name (or use the clone URL shown on the repository page).
