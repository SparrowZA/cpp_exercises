2. Open UDP ports   (~20 lines)
Nmap is a software tool used by hackers, good and bad. It searches the network and returns 
a list of IP address, protocols, port numbers and other facts that may be interesting to a 
hacker. The list in the csvEx.py is an example of the output of a nmap scan.
Create a script that imports the nmapOutput list from the csvEx.py file. It should then 
create a new list of open udp ports and print the new list on the command line.

Note: * .split(delimiter) will be useful

** The format of the semi-colon seperated string elements are as follows:
    "IP-address;    protocol;   port number;    description;    status"
     192.168.2.43;  tcp;        22;             ssh;            open

Your output should look like this:
['192.168.9.23;udp;9999;rpcbind;open', '192.168.9.80;udp;53;domain;open', '192.168.9.80;udp;7912;;open', '192.168.9.80;udp;9822;;open']
